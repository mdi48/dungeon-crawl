# Dungeon Crawl

A simple cli adventure game made as a beginner's project for learning Elixir (I used Learn Functional Programming with Elixir by Ulisses Almeida, and I found it to be very helpful even if it is out-of-date). This is not meant to be much more than a simple demonstration of what I made during my time spent reading the book and putting my knowledge to practice, so do not be surprised if it is rather simplistic. 

To run it, perform the following commands in the root folder of the project (after having pulled it of course):
```bash
mix do deps.get + deps.compile 
mix start
```
and the game will start. Choose from one of the three available hero classes:

- Knight: Higher HP, lower damage. Good for those who are cautious.
- Wizard: Lower HP, higher damage. Good for those who enjoy glass cannons.
- Rogue: In between the other two in terms of HP and damage. Good for those seeking a balanced approach.

You will then be introduced into the fearsome dungeon full of deathtraps and nightmarish monsters... can you survive to see the sunlight again?
